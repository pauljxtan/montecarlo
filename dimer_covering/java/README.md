# Dimer covering: Java implementation #

### Usage ###

```bash
$ java Main [input file]
```

The input file should consist of a single line with the following values in order:
* Lattice width
* Lattice height
* Initial temperature
* Final temperature
* Time constant (tau)
* Frameskip
* Verbose flag (0 or 1)

e.g. "100 100 10.0 1e-2 1e4 10 1"
